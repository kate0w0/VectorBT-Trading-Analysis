# 📉 Dual Moving Average Crossover: SNGSP Analysis 📈
Цель: исследование торговой стратегии на основе пересечения скользящих средних для привилегированных акций Сургутнефтегаза (SNGSP)
## Данные
- Часовые свечи по SNGSP
- Период: 2024 год
- Источник: Московская биржа (API MOEX)

Выбор SNGSP обусловлен высокой ликвидностью и трендовостью актива, что критично для стратегий следования за трендом. Период 2024 года выбран для тестирования модели в условиях повышенной рыночной волатильности и проверки эффективности алгоритмов риск-менеджмента на актуальных данных

## Стратегия
В основе лежит стратегия пересечения двух скользящих средних:
- **Сигнал на покупку** возникает, когда «быстрая» скользящая средняя пересекает «медленную» **снизу вверх**. Это указывает на начало восходящего тренда.
- **Сигнал на продажу** возникает, когда «быстрая» скользящая средняя пересекает «медленную» **сверху вниз**. Это сигнализирует о смене тренда на нисходящий и необходимости закрыть позицию
### Риск-менеджмент (Stop Loss)
Для защиты капитала от резких обвалов рынка в стратегию встроен Stop Loss (5%):
- Если после покупки цена акции падает на 5% от уровня входа, позиция закрывается автоматически, не дожидаясь обратного пересечения скользящих средних.

Это критически важный элемент для акций SNGSP, которые могут быть подвержены резким рыночным колебаниям

## Процесс оптимизации
В проекте не просто выбраны случайные числа для средних, а проведен Grid Search (поиск по сетке)

Цель оптимизации: Максимизация итоговой прибыли (Total Profit)

## Результат
Итоговые показатели оптимизированной стратегии для акций SNGSP:
- Оптимальные параметры: Быстрая MA (111) / Медленная MA (116)
- Общее количество сделок: 89
- Сделки на покупку: 45
- Сделки на продажу: 44
- Максимальная чистая прибыль: ~513 791.80 руб. (при начальном капитале 1 000 000 руб.)
- Доходность: более 51% годовых

## Итог
Проект демонстрирует, что использование автоматизированного бэктестинга и оптимизации параметров позволяет найти настройки, при которых классическая трендовая стратегия остается прибыльной даже на волатильном рынке 2024 года. Внедрение Stop Loss (5%) стало ключевым фактором сохранения капитала, позволяя системе вовремя выходить из актива в моменты резких коррекций, не дожидаясь разворота скользящих средних
